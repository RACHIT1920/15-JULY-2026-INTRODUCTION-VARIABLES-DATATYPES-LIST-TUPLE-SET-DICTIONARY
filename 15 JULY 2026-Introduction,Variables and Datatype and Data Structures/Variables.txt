Data Structures
A. List
->It is an ordered,mutable collection of items. It can hold multiple values including different data types
->methods:pop,append,extend,copy,remove,count,index,etc
# =========================
# PYTHON LIST METHODS
# =========================

# 1. append(x)
# Adds an element at the end of the list.
lst.append(x)

# 2. extend(iterable)
# Adds all elements of another iterable to the list.
lst.extend(iterable)

# 3. insert(index, element)
# Inserts an element at the specified index.
lst.insert(index, element)

# 4. remove(element)
# Removes the first occurrence of the specified element.
lst.remove(element)

# 5. pop(index=-1)
# Removes and returns the element at the given index.
# Default removes the last element.
lst.pop()
lst.pop(index)

# 6. clear()
# Removes all elements from the list.
lst.clear()

# 7. index(element, start=0, end=len(list))
# Returns the index of the first occurrence of the element.
lst.index(element)
lst.index(element, start, end)

# 8. count(element)
# Returns the number of occurrences of an element.
lst.count(element)

# 9. sort()
# Sorts the list in ascending order.
lst.sort()

# Sort in descending order.
lst.sort(reverse=True)

# Sort using a custom key.
lst.sort(key=len)

# 10. reverse()
# Reverses the list in place.
lst.reverse()

# 11. copy()
# Returns a shallow copy of the list.
new_list = lst.copy()
